# 🎮 Escape From Duckov - フィードバック

[Escape From Duckov](https://escapefromduckov.net) 公式フィードバックリポジトリへようこそ！

このリポジトリは、ゲーム情報サイトのユーザーフィードバック、バグレポート、機能リクエスト、データ更新リクエストの収集に特化しています。

## 📝 フィードバックの送信方法

すべてのフィードバックを追跡するために GitHub Issues を使用しています。新しい Issue を作成する際は、適切なテンプレートをお選びください：

### 🐛 バグレポート
バグや正しく動作していない機能を見つけましたか？[バグレポートを作成](../../issues/new?template=bug_report.yml)

### ✨ 機能リクエスト
新機能や改善のアイデアがありますか？[機能リクエストを送信](../../issues/new?template=feature_request.yml)

### 📊 データ更新リクエスト
ゲームデータの更新や修正が必要ですか？[データ更新をリクエスト](../../issues/new?template=data_update.yml)

### 💬 その他のフィードバック
他に共有したいことがありますか？[一般的なフィードバック Issue を開く](../../issues/new?template=other.yml)

## 🌐 対応言語

## 📋 送信前のチェック

新しい Issue を作成する前に、以下をご確認ください：
1. **既存の Issues を検索** - 重複を避けるため
2. **ウェブサイトを確認** - 問題がまだ存在するか確認
3. **明確な詳細を提供** - 問題を理解し対処するために

## 🏷️ ラベル

Issues を整理するために以下のラベルを使用しています：

- `bug` - 正しく動作していないもの
- `feature` - 新機能または改善リクエスト
- `data-update` - ゲームデータの更新または修正リクエスト
- `other` - 一般的なフィードバックまたは質問
- `priority-high` - 早急な対応が必要な問題
- `priority-medium` - 重要な問題
- `priority-low` - あると便利なもの
- `in-progress` - 現在作業中
- `resolved` - 問題が解決済み
- `wontfix` - 対応しない問題
- `duplicate` - 他の Issue と重複
