# 🎮 Escape From Duckov - 反馈收集

欢迎来到 [Escape From Duckov](https://escapefromduckov.net) 官方反馈仓库！

本仓库专门用于收集我们游戏资讯网站的用户反馈、Bug 报告、功能请求和数据更新请求。

## 📝 如何提交反馈

我们使用 GitHub Issues 来跟踪所有反馈。创建新 Issue 时，请选择适当的模板：

### 🐛 Bug 报告
发现了 Bug 或功能异常？[创建 Bug 报告](../../issues/new?template=bug_report_zh.yml)

### ✨ 功能请求
有新功能或改进的想法？[提交功能请求](../../issues/new?template=feature_request_zh.yml)

### 📊 数据更新请求
需要更新或修正游戏数据？[请求数据更新](../../issues/new?template=data_update_zh.yml)

### 💬 其他反馈
有其他想分享的内容？[提交其他反馈](../../issues/new?template=other_zh.yml)

### 🌐 英文模板
如果您更习惯使用英文，也可以使用 [英文模板](../../issues/new/choose) 提交反馈。

## 🌐 支持的语言

您可以使用以下任何语言提交反馈：
- 🇺🇸 English（英语）
- 🇨🇳 中文
- 🇯🇵 日本語（日语）
- 🇷🇺 Русский（俄语）

## 📋 提交前须知

在创建新 Issue 之前，请：
1. **搜索现有 Issues** 避免重复提交
2. **检查网站** 确认问题仍然存在
3. **提供清晰的详细信息** 帮助我们理解并解决您的反馈

## 🏷️ 标签说明

我们使用以下标签来组织 Issues：

- `bug` - 网站功能异常
- `feature` - 新功能或改进请求
- `data-update` - 游戏数据更新或修正请求
- `other` - 一般反馈或问题
- `priority-high` - 需要立即处理的紧急问题
- `priority-medium` - 重要问题
- `priority-low` - 可选功能
- `in-progress` - 正在处理中
- `resolved` - 问题已解决
- `wontfix` - 不会处理的问题
- `duplicate` - 重复的 Issue
